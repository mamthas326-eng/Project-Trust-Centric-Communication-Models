# Trust-Centric IoT Communication Model

## Description
This project implements a trust-based communication model using NS-3.

## Components
- NS-3 simulation (AODV)
- Trust evaluation
- Performance metrics (PDR, Delay, Throughput)
- Graph analysis

## Tools Used
- NS-3
- NetAnim
- Python (Matplotlib)

## Output
- Simulation results
- Graphs
- Network animation
